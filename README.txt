
# Лабораторный анализатор точности измерений

Проект представляет собой веб-приложение для анализа точности измерений в соответствии с ГОСТ Р ИСО 5725-2-2002. Приложение позволяет рассчитывать показатели повторяемости, воспроизводимости и правильности на основе загруженных данных.

## Основные возможности

- Расчет метрик точности измерений:
  - Повторяемость (repeatability)
  - Воспроизводимость (reproducibility)
  - Правильность (trueness)
- Визуализация данных:
  - Гистограммы распределения
  - Боксплоты для сравнения лабораторий
  - Контрольные карты
- Экспорт результатов в формате Excel
- API для интеграции с другими системами

## Технологии

- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript, Plotly
- **Библиотеки**: Pandas, NumPy, SciPy, XlsxWriter
- **Формат данных**: Excel (.xlsx, .xls)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repo/measurement-accuracy-analyzer.git
   cd measurement-accuracy-analyzer
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` с переменными окружения:
   ```
   SECRET_KEY=your-secret-key-here
   ```

4. Запустите приложение:
   ```bash
   python app.py
   ```

   Приложение будет доступно по адресу: `http://localhost:5001`

## Использование

1. Загрузите файл Excel с данными измерений через веб-интерфейс
2. Выберите необходимые расчеты
3. Просмотрите результаты и графики
4. Экспортируйте результаты при необходимости

### Формат данных

Файл Excel должен содержать следующие столбцы (в зависимости от требуемых расчетов):

- **Для всех расчетов**: `Data` - результаты измерений
- **Для воспроизводимости**: `Laboratory` - идентификатор лаборатории
- **Для временных графиков**: `Date` или `Order` - дата или порядковый номер измерения

## API документация

Приложение предоставляет REST API для программного доступа к функционалу. Документация доступна по адресу `/api/docs` после запуска приложения.

Для использования API требуется ключ аутентификации, который передается в заголовке `X-API-KEY`.
